1.The "Main.vi" is the main UI and all the testing are done using this VI.
2.The "SubpanelTests" library contains the list of test VIs.
3.The Test VIs can be executed individually and also in collective manner.
4.Open the Project file for the particular example.
5.In the toolbar navigate to Tools >> Caraya >> Run Tests in Active Project.
6.This action will run all the tests in the project and generates a test report.
